MODELO CLIENTE servidor
Es un modelo que se usa para acceder a internet y obtener recursos en informacion
Cliente y servidor saben el tipo de mensaje que necesitan. 

El protocolo HTTP define el formato de los mensajes. Son reglas para transmitir información entre dispositivos
Hypertext transfer protocol

Los metodos HTTP pueden ser sustantivos, o verbos http que indican la intencion de la solicitud
